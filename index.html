Course Catalog Description

This course will cover the ceation of Web pages and sites using HTML, CSS, Javascrit and graohical applications. Networked multimedia distribution technologies are also explored. The design of effective Website including page layout, user interface design, graphic design and content flow and site structure as well as managment of Website resources inclduing intanet management and design considerations are addressed. Students design and create a major Website with multiple pages and cross linked strcutures. Prerequisites: [none]Credit: 2-2-3 Semester Hours

Course Outcomes:

Students completing this course will be able to:

  • Recognize HTML, CSS, and Javascript mark-up nad code in a web page/application.
  • Select the proper mark-up tags or code to acheieve a particular result.
  • Identify improperly used mark-up and code
  • Produce modern standards compalint web pages
  • Deploy web pages to a public server

Course Objectives:

At the conclusion of this course, each successful student will be able to:

  • Explain the client and server architecture of the Internet and related web technologies
  • Use a basic text editor and other software tools to create web pages using HTML, CSS, Javascript, jQuery, and images
  • Deploy web pages to a Linux-based web server using sftp
